# Employee Management System

## Overview
The Employee Management System is a Java-based application that manages employee records using JDBC and Oracle Database.

## Features
- Add Employee
- Update Employee
- View Employee Details
- View All Employees
- Delete Employee

## Technologies Used
- Java
- JDBC
- Oracle Database
- DAO Design Pattern

## Project Structure
- bean/
  - Employee.java
- dao/
  - EmployeeDAO.java
- service/
  - EmployeeService.java
- util/
  - DBUtil.java
- main/
  - TestEmployee.java

## CRUD Operations
- Create Employee
- Read Employee
- Update Employee
- Delete Employee
